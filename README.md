# 📊 Student Score Prediction

## 📌 Project Overview

This project uses Machine Learning to predict student scores based on study hours. A Linear Regression model is built to understand the relationship between study time and performance.

---

## 🎯 Objective

* Predict student scores using study hours
* Understand basic Machine Learning concepts
* Visualize prediction results

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Scikit-learn

---

## 📂 Dataset

The dataset contains:

* Study Hours
* Student Scores

---

## 📈 Model Used

* **Linear Regression**

---

## 📊 Results

* Built a regression model to predict scores
* Visualized actual vs predicted values
* Achieved good accuracy on dataset

---

## 🚀 How to Run

1. Install libraries:
   pip install pandas matplotlib scikit-learn

2. Run the script:
   python student_prediction.py

---

## 📊 Key Features
- Data preprocessing using pandas
- Model building using Linear Regression
- Data visualization using matplotlib
- Predicts score based on study hours
